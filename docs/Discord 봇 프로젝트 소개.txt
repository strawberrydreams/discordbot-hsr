# discord_bot_project

Windows에 파이썬 설치, pip 설치, discord.py 라이브러리 설치, SSL 인증서 설치, 코드 포팅, 봇 토큰 + 채널 ID 확인
.env 파일에 실제 Discord Bot Token이 저장되어 있음
.env 파일을 이용해서 Token을 불러오려면 dotenv 라이브러리 설치해야 함
서버에 올려서 환경 변수로 Token을 등록하면 dotenv 대신 다른 코드를 써야 함
Windows cmd 명령어와 Mac 터미널 명령어가 서로 다름

채널 및 역할

***Information 카테고리 (관리자는 쓰기 가능, 일반 멤버는 읽기만 가능) (이모지 반응 포함)

1. 규칙: 땅끝소초 서버를 이용하기 전 반드시 읽어주세요.
2. 공지: 땅끝소초 서버의 중요한 공지사항이 업로드됩니다.
3. 새친구: 땅끝소초 서버의 신규 유저를 모두 환영해 주세요~
4. 이벤트: 땅끝소초 서버의 이벤트를 홍보하는 공간입니다.
5. 관리자-문의: 땅끝소초 서버에 대한 문의사항, 제안, 요청을 관리자에게 전달할 수 있습니다.


***Public 카테고리 (모두 읽고 쓰기 가능)
1. 채팅-채널: 땅끝소초 서버 규칙을 준수하는 모든 주제의 토론, 대화가 허용됩니다.
2. 모집-채널
3. 팬아트-채널: 팬아트 업로드 시 원작자 또는 출처를 명시해주세요. (생성형 AI 작품 허용)
4. 창작-채널: 창작물 업로드 시 본인임을 명시해주세요. (생성형 AI 작품 허용)
5. 뉴스-채널


***Private 카테고리 (관리자 전용)
1. 프라이빗-라운지: 선택받은 자만이 이 비밀 공간의 조명을 받습니다.
2. 업데이트: 땅끝소초 서버의 중요한 업데이트 내역이 업로드됩니다.
3. 명령어: Discord Bot의 명령어를 실험하고 테스트할 수 있는 공간입니다.
4. 실험실: Discord의 개발자 모드 기능을 실험하고 테스트할 수 있는 공간입니다.
